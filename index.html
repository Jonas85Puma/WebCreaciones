+

Una landing page simple: 5-7 días. Sitio web básico: 1-2 semanas. Te mantengo informado del progreso cada día.

+

Trabajo con código básico pero funcional. No uso plantillas complicadas, sino que creo algo sencillo y personalizado para ti.

+

No incluyo hosting ni dominio, pero te ayudo a elegir opciones económicas y te explico cómo configurar todo paso a paso.

+
